NFCKING – én lenke, ett kort, uendelig fleksibelt.

Lag og del digitale visittkort med NFC/QR. Bedrifter kan administrere ansatte, logoer, farger og lenker på sekunder – alt vises live.

  • ✅ Forhåndsvisning av digitalt kort (live)
  • ✅ Matchende fysisk kort-mockup (red / light / dark)
  • ✅ Ubegrensa lenker med tekst over/under
  • ✅ Klar for NFC/QR og GitHub-publisering

Mockupen bruker samme renderer som demoen nedenfor.

Hvordan funker det?

1) Fyll inn profil

Legg inn data og last opp logo.

2) Se live-forhåndsvisning

Digitalt og fysisk kort rendres identisk.

3) Publiser

Trykk “Publiser” i admin for å laste opp til GitHub Pages.

4) Skaler

Administrer flere ansatte og design på samme system.
